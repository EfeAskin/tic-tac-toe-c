# Tic Tac Toe - C Console Game

Bu proje, C dilinde yazılmış bir konsol tabanlı **Tic Tac Toe (X ve O)** oyunudur. İki oyuncu sırayla hamle yapar ve kazanmayı hedefler. Oyun berabere bittiğinde kullanıcıya bildirilir ve tekrar oynama seçeneği sunulur.

## Özellikler
- 3x3 oyun tahtası
- İki oyunculu mod (X ve O)
- Kazanan ve beraberlik kontrolü
- Tekrar oynama seçeneği

## Kurulum ve Çalıştırma
1. Kod dosyasını indirin (`tic_tac_toe.c`).
2. Terminal veya komut istemcisinde derleyin:
   ```bash
   gcc tic_tac_toe.c -o tic_tac_toe
